== Description ==

<strong>HeBo Logo Slider</strong>

## Features

- Add logo sliders
- Support variable width logos
- Support linkable logos
- Uses slick.js 1.9.0

== Installation ==

1. Install the HeBo Logo Slider either via the WordPress plugin directory or by uploading the files to your server at wp-content/plugins.

== Frequently Asked Questions ==

= Can the logos be linked to internal or external resources? =

Yes

= Are there any limits in the amount of logos? =

No

= Is the design customizeable? =

Yes, simple override the slider styles within your used theme.
